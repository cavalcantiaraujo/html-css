Hierarquia de títulos

Como criar sub títulos

Para criar sub títulos devemos usar as tags de h1 a h6.

Principais formatações

Negrito / Dstaque

Para deixar algo em negrito devemos usar a tag strong, dessa forma damos sentido a palavra, dizemos que ela é forte, que tem um valor semântico

Itálico / Ênfase

Para deixar o termo em itálico utilizamos a tag em.

Texto marcado

Para deixar algo marcado como um marca-texto podemos utilizar a tag MARK. Podemos alterar a cor da marcação utilizando a tag Style. Pode ser utilizada somente na marcação, ou no head para que seja alterado as cores de todas as marcações.

Texto grande e texto pequeno

Para deixar um texto grande existe a tag BIG porém ela não é mais utilizada, pois não se trata de semântica e sim de estilo. Já para deixar o texto pequeno é possível dentro da semântica, pois pode ser utilizado para documentos.

Texto deletado

Podemos carcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
